追求简洁、美观和小巧的导航页生成器。该项目还在实验阶段。

## 定位

优点:

- 小巧: 不使用第三方框架，在经过EJS模板编译后只有HTML和CSS
- 美观: 在做到上一条的前提下，尽可能追求美观

缺点:

- 需要现代浏览器(Chromium 92+, Firefox 105+, Safari 16+)
